# AgilexRobotPythonFramework
python-can wrapped framework to parse agilex can-bus data .
 |no |  tested on |  
 |:-:| :-------------------: | 
 | 1 | scout mini |
 
  requirements:
<pre>
  sudo apt-get install net-tools
  pip3 install python-can
  pip3 install simple-pid
  pip3 install matplotlib
  pip3 install numpy
</pre>
